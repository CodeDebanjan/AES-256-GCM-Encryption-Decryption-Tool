## 📌 Description

This project is a file encryption and decryption tool designed to help users securely protect their files using password-based encryption.

Users can:

- Upload any file
- Encrypt it with a password
- Download the encrypted file
- Later decrypt the file using the exact same password

The application ensures that only users with the correct password can recover the original file.

«⚠️ Important: If the password is forgotten, the encrypted file cannot be decrypted.»

## 🚀 How to Use

### 🔐 Encryption Process

1. Upload File

Click the Upload File button and select the file you want to encrypt from your device.

2. Enter Password

Type a strong password into the password field.

«⚠️ Remember this password carefully. You will need the exact same password to decrypt the file later.»

3. Encrypt File

Click the Encrypt button to start the encryption process.

The software will securely encrypt your file using the provided password.

4. Download Encrypted File

Once encryption is complete, the encrypted file will be available for download.

Save the encrypted file safely.

---

### 🔓 Decryption Process

1. Upload Encrypted File

Click the Upload Encrypted File button and select the encrypted file.

2. Enter Password

Enter the exact same password that was used during encryption.

«⚠️ If the password is incorrect, the file cannot be decrypted properly.»

3. Decrypt File

Click the Decrypt button to begin the decryption process.

The software will verify the password and recover the original file.

4. Download Original File

After successful decryption, the original file will be available for download.

## 🔐 Security Notes

- Losing the password may result in permanent loss of access to the encrypted file.
